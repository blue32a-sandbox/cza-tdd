整数閉区間を示すクラス（あるいは構造体）をつくりたい。
整数閉区間オブジェクトは下端点と上端点を持ち、文字列表現も返せる
（例: 下端点 3, 上端点 8 の整数閉区間の文字列表記は "[3,8]"）。
ただし、上端点より下端点が大きい閉区間を作ることはできない。
整数の閉区間は指定した整数を含むかどうかを判定できる。
また、別の閉区間と等価かどうかや、完全に含まれるかどうかも判定できる。

- [x] 整数閉区間を示すクラスをつくる

- [x] 整数閉区間が持つ下端点 (lower endpoint)を取得できる
    - [x] 整数閉区間が下端点 3を持つとき3を返す
    - [x] 整数閉区間が下端点 4を持つとき4を返す
- [x] 整数閉区間が持つ上端点 (upper endpoint) を取得できる
    - [x] 整数閉区間が上端点 8を持つとき8を返す
    - [x] 整数閉区間が上端点 9を持つとき9を返す

- [x] 整数閉区間の下端点は上端点より小さい
    - [x] 下端点が上端点より大きいと例外が発生する

- [x] 整数閉区間は文字列表現を返すことができる
    - [x] 下端点 3、上端点 8の整数閉区間は"[3,8]"を返す

- [x] 整数閉区間に指定した整数を含むかどうかを判定できる
    - [x] 整数閉区間 下端点 3、上端点 8の時、3はtrueを返す
    - [x] 整数閉区間 下端点 3、上端点 8の時、8はtrueを返す
    - [x] 整数閉区間 下端点 3、上端点 8の時、2はfalseを返す
    - [x] 整数閉区間 下端点 3、上端点 8の時、9はfalseを返す

- [x] ある整数閉区間が別の整数閉区間と等価かどうかを判定できる
    - [x] 整数閉区間A(3～8)と整数閉区間B（3～8）は等価
    - [x] 整数閉区間A(3～8)と整数閉区間B（2～8）は等価ではない
    - [x] 整数閉区間A(3～8)と整数閉区間B（3～9）は等価ではない

- [x] ある整数閉区間に別の整数閉区間が完全に含まれるかどうかを判定できる
    - [x] 整数閉区間A(3～8）に整数閉区間B（3～8）は完全に含まれる
    - [x] 整数閉区間A(3～8）に整数閉区間B（2～8）は完全に含まれない
    - [x] 整数閉区間A(3～8）に整数閉区間B（3～9）は完全に含まれない
    - [x] 整数閉区間A(3～8）に整数閉区間B（2～9）は完全に含まれない
